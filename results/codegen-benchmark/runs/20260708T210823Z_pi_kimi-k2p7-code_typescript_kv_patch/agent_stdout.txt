Created in the current directory:

- `solution.ts` – TypeScript implementation
- `run.sh` – POSIX shell script that compiles (if needed) with `tsc` and runs with `node`

`./run.sh` reads JSON Lines from stdin, applies timestamped `set`/`delete` operations in ascending `ts` order (stable for ties), then prints the final map as `key=value` lines sorted bytewise.

Verified with a few quick cases, including empty input, tie ordering, and deletion of absent keys.
