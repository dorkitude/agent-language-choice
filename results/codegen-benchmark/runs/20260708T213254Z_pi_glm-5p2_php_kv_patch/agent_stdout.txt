## Summary

**`run.sh`** — POSIX shell wrapper that execs `php main.php`.

**`main.php`** — does the work using only PHP's standard library:
1. Reads stdin line by line, skipping blank/malformed lines, tagging each record with its input index (`__idx`).
2. **Stable sort** by `ts` ascending — since PHP's `usort` isn't guaranteed stable, the original input index is used as a tiebreaker so tied timestamps preserve input order.
3. Applies operations in sorted order: `set` stores the value (defaulting to `''` if missing), `delete` removes the key (no-op if absent).
4. Sorts final keys **bytewise** via `ksort(..., SORT_STRING)` (strcmp = byte-by-byte).
5. Prints `key=value` lines; empty map → no output.

### Verified behaviors
- ✅ Ascending `ts` ordering, with ties preserving input order
- ✅ Bytewise key sort (uppercase `A,B` before lowercase `a,b`)
- ✅ Empty input and fully-deleted map → no output
- ✅ Delete on missing key is a no-op
- ✅ Negative timestamps and out-of-order input
- ✅ Values containing `=` preserved verbatim
- ✅ Blank lines skipped

Deterministic, no network, standard library only. 🐘
