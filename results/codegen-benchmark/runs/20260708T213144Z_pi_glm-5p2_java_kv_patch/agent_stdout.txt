## Files

**`Main.java`** — pure Java stdlib implementation:
- A tiny hand-rolled recursive-descent JSON parser (objects, arrays, strings w/ all standard escapes incl. `\uXXXX`, numbers, bools, null) — since Java has no built-in JSON.
- Parses each JSONL line into a `Record` keeping its original input index for tie-breaking.
- **Stable sort** by `ts` ascending (Java's `List.sort` is stable; the explicit `order` field makes it bulletproof), so tied timestamps preserve input order.
- Applies `set`/`delete` onto a `TreeMap`.
- **Bytewise UTF-8 key ordering** comparator — this equals code-point lexicographic order, satisfying "bytewise/lexicographic" for both ASCII and non-ASCII keys.
- Empty map → prints nothing.

**`run.sh`** — POSIX sh that compiles with `javac` then runs `java Main`, piping stdin→stdout.
