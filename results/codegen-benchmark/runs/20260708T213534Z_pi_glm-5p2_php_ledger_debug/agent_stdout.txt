## Summary

The buggy CLI had one critical bug: it **overwrote** each account's balance instead of **accumulating** it.

```php
// BUG: overwrites on every row
$balances[$parts[0]] = intval($parts[1]);

// FIX: accumulate per account
$balances[$parts[0]] = ($balances[$parts[0]] ?? 0) + intval($parts[1]);
```

I also hardened the line-ending trim from `rtrim($line, "\n")` → `rtrim($line, "\r\n")` so CRLF inputs don't poison the delta field (the `\r` would have failed the integer regex).

### Changes
- **`main.php`** — the two fixes above; `run.sh` left untouched (it already honors the `./run.sh` contract by calling `php main.php`).

### Verification
