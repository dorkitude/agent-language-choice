The D&D REST API is wired up through a Vite dev-server middleware plugin.

What’s in place:

- `src/dndApiPlugin.ts` — implements all required endpoints:
  - `GET /health`
  - `POST /v1/dice/stats`
  - `POST /v1/checks/ability`
  - `POST /v1/encounters/adjusted-xp`
  - `POST /v1/initiative/order`
- `vite.config.ts` — loads the plugin via `defineConfig`.
- `run.sh` — starts Vite bound to `127.0.0.1` on the `PORT` env var.

The server uses the exact pinned Vite 8.1.3 / TypeScript versions in the workspace. No extra server framework was added, and I didn’t start the server.
