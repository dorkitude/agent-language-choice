I implemented the full D&D REST API in `app.py` using **Flask 3.1.3** and **Python 3.14.6**, and kept the existing `run.sh`.

**Implemented endpoints:**
- `GET /health` → `{"ok": true}`
- `POST /v1/dice/stats` — parses `XdY±Z`, returns dice stats, min/max/average, 400 on invalid expressions
- `POST /v1/checks/ability` — ability check with total, success, margin
- `POST /v1/encounters/adjusted-xp` — CR-to-XP table, monster count multiplier, summed level-3 thresholds, difficulty rating
- `POST /v1/initiative/order` — score = roll + dex, sorted by score desc, dex desc, name asc

`./run.sh` starts the server in the foreground on `127.0.0.1:$PORT` using the pre-installed `.deps` Flask. I didn’t start the server.
