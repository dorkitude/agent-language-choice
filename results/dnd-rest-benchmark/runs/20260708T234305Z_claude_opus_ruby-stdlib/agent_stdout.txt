The server is ready. Here's what I built:

**`server.rb`** — a pure Ruby 4.0.5 stdlib HTTP server (just `socket` + `json`, no gems, no Sinatra/Rack):

- 🩺 `GET /health` → `{"ok":true}`
- 🎲 `POST /v1/dice/stats` — regex-parsed `<count>d<sides>[±mod]` grammar, positive count/sides enforced, 400 on invalid
- ⚔️ `POST /v1/checks/ability` — total/success/margin
- 👹 `POST /v1/encounters/adjusted-xp` — CR→XP table, count multipliers, summed level-3 thresholds, difficulty tiering
- 🏃 `POST /v1/initiative/order` — sorts by score↓, dex↓, name↑

Smoke-tested every endpoint against the spec's sample requests — all outputs match byte-for-byte, and bad expressions return 400. 

A few deliberate touches:
- `numify` keeps whole-number values as JSON integers (`multiplier: 2`, not `2.0`) to match the spec examples 🔢
- Each connection handled in its own thread with `Connection: close`
- Listens on `127.0.0.1` using `PORT` (defaults to 8080)

`run.sh` already invokes `ruby server.rb` in the foreground, so the contract is satisfied. I did **not** start the server (killed my test instance). Ready to go! 🚀
