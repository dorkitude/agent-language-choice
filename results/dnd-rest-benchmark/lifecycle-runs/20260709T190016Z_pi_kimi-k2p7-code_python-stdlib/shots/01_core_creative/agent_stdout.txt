Chef, the core D&D REST engine is ready! 🎲✨

I created `server.py` using only Python 3.14.6 standard library (`http.server`, `json`, `re`, `os`) and updated `./run.sh` to `exec python3 server.py` so the server runs directly in the foreground on `127.0.0.1:$PORT`.

Implemented endpoints:

- `GET /health` → `{"ok": true}`
- `POST /v1/dice/stats` → parses `XdY±Z`, returns stats including the exact mean (integer when whole, `.5` when half)
- `POST /v1/checks/ability` → total, success, margin
- `POST /v1/encounters/adjusted-xp` → XP, multiplier, thresholds, and difficulty
- `POST /v1/initiative/order` → score, sorted by score desc, dex desc, name asc

All sample responses matched exactly when I spun it up briefly to verify. No server is running now, and `./run.sh` is good to go! 🚀
