What I built (`server.py`, pure Python 3.14 stdlib — `http.server` + `json` + `re`):

- **`GET /health`** → `{"ok": true}`
- **`POST /v1/dice/stats`** — regex-parsed `<count>d<sides>[±mod]`, positive count/sides enforced, invalid → 400
- **`POST /v1/checks/ability`** — total/success/margin
- **`POST /v1/encounters/adjusted-xp`** — CR→XP table, count multipliers, summed party thresholds, highest-difficulty-reached
- **`POST /v1/initiative/order`** — sort by score desc, dex desc, name asc

A couple of deliberate touches: integer-vs-float normalization (`_num`) so `multiplier` comes back as `2` not `2.0` while `1.5` stays a float, and `ThreadingHTTPServer` with `HTTP/1.1` + `Content-Length` for well-behaved keep-alive connections. All five sample requests match the spec byte-for-byte. 🐉
