The server is implemented in `./server.py` using only Python 3.14.6 stdlib (`http.server`, `json`, `re`, `os`), and `./run.sh` starts it in the foreground.

I verified the four endpoints plus `/health` all return the expected responses from the spec. The server listens on `127.0.0.1` and uses the `PORT` environment variable (defaults to `8080`).

Key choices:
- Dice stats average is integer `(min + max) // 2` to match the sample.
- Encounter thresholds are summed per party member and difficulty is the highest threshold reached (or `trivial`).
- Initiative ties break by `dex` descending, then `name` ascending.
