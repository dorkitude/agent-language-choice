## What I added to `app.rb`

**Schema** (`Store`) — two new SQLite tables `monsters` and `items`, created idempotently in `init_schema!` and cleaned up in `reset!` alongside the existing tables.

**`Compendium` module** — domain logic with `Duplicate` (→409) and `NotFound` (→404) exceptions, plus `ArgumentError` for malformed input (→400):
- `create_monster` / `read_monster` — tags stored as JSON; create response omits `tags`, read response includes them.
- `create_item` / `read_item` — identical create/read shape.

**Routes + controller actions** — `POST/GET /v1/compendium/monsters[/:slug]` and `.../items[/:slug]`.
