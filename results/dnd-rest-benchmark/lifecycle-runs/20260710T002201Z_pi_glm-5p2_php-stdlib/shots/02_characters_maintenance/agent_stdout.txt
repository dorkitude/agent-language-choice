## Changes

**`index.php`** — added two helper functions and three new POST routes, with no changes to existing endpoints:

- **`ability_modifier(int $score): int`** — `(int) floor(($score - 10) / 2)`. PHP's `floor()` rounds toward −∞, so score 9 → `floor(-0.5)` → **−1** ✓
- **`proficiency_bonus(int $level): int`** — explicit `match(true)` tier table (1-4→2, 5-8→3, 9-12→4, 13-16→5, 17-20→6)
- **`POST /v1/characters/ability-modifier`** — validates `score` is an int in 1–30 (rejects floats/strings/missing/out-of-range with `400`)
- **`POST /v1/characters/proficiency`** — validates `level` is an int in 1–20 (else `400`)
- **`POST /v1/characters/derived-stats`** — validates `level` (1–20), all six abilities (1–30 each), and `armor` (`base`/`dex_cap` ints), then computes `hp_max = level * (6 + con_mod)`, `armor_class = base + min(dex_mod, dex_cap) + shield_bonus` (shield = 2 when `armor.shield === true`)
