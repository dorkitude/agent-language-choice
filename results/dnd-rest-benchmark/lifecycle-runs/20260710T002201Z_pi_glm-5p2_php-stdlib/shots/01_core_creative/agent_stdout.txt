**Key design choices:**
- `average` uses exact arithmetic `(min+max)/2` ‚Äî PHP's `/` yields int when evenly divisible (‚Üí `10`), float otherwise (‚Üí `3.5`), and `json_encode` serializes whole floats without decimals.
- `multiplier` stays int (`1`,`2`,`3`,`4`) or float (`1.5`,`2.5`) naturally, matching the spec's mixed output.
- CR lookup normalizes to string and uses `array_key_exists` (handles both int/string CR keys); unknown CR ‚Üí 400.
- All responses set `Content-Type: application/json`; unknown routes ‚Üí 404.
