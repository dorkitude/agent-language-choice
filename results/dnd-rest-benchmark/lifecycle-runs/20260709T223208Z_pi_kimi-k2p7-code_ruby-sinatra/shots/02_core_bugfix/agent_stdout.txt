Fixed the dice-stats average so it returns a float for odd sums instead of truncating to an integer.

- File: `app.rb`
  - Changed:
    ```ruby
    average = (min + max) / 2
    ```
    to:
    ```ruby
    average = (min + max).even? ? (min + max) / 2 : (min + max) / 2.0
    ```

This preserves the integer response for `2d6+3` (`average: 10`) and correctly returns `9.5` for `1d20-1`.

Syntax check passed, and `./run.sh` is already in place. Ready for the evaluator, Chef.
