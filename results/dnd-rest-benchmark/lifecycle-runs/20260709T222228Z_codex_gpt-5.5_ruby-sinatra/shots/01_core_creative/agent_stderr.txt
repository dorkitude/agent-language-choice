<count>d<sides>[+<modifier>|-<modifier>]

Response fields:

```json

  "sides": 6,
  "modifier": 3,
  "min": 5,
  "max": 15,
  "average": 10
}
```

Request:

```json
{"roll": 9, "modifier": 5, "dc": 15}
```

Rules:

- `total = roll + modifier`
- `success = total >= dc`
- `margin = total - dc`

Response:

```json
{"total": 14, "success": false, "margin": -1}
```
