I implemented the core D&D REST API as Next.js 16.2.10 route handlers under `app/`:

- `GET /health` → `{ "ok": true }`
- `POST /v1/dice/stats` → parses `XdY±M` expressions, returns count, sides, modifier, min, max, average
- `POST /v1/checks/ability` → total, success, margin
- `POST /v1/encounters/adjusted-xp` → CR/XP table, multiplier, difficulty, thresholds
- `POST /v1/initiative/order` → score, dex, name tie-breaker sorting

I also added a minimal `app/layout.tsx` (required by the App Router) and updated `run.sh` to run in the foreground on `127.0.0.1:$PORT` using the local Next.js binary:

```bash
#!/usr/bin/env bash
set -euo pipefail
exec npx next dev -H 127.0.0.1 -p "$PORT"
```

All endpoints returned the expected JSON during a quick smoke test. `./run.sh` is ready! 🚀
