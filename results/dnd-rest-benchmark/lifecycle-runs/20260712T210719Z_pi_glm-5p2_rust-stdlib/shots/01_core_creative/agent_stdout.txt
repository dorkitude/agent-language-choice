## What was built

A pure-stdlib Rust HTTP server in `src/main.rs` (no Cargo deps, no HTTP/JSON crates) — exactly matching the target guidance:

- **HTTP layer**: `std::net::TcpListener`/`TcpStream` + `BufReader`, reads the request line, headers, and full body via `Content-Length` (with `Expect: 100-continue` support), thread-per-connection for concurrency.
- **JSON layer**: hand-rolled recursive-descent parser (objects, arrays, numbers, strings w/ escapes + surrogate pairs, bools, null) and a serializer with smart number formatting (integers print without `.0`, `.5` fractions preserved).

## Edge cases verified ✅
- Dice: negative modifiers, fractional averages (`1d6`→`3.5`), whitespace trimming, strict validation (`0d6`, `2d6+-3`, `1/2`, missing fields → 400).
- Encounters: fractional CRs (`1/8`,`1/4`,`1/2`), `1.5`/`2.5` multipliers, all 5 difficulty tiers incl. `trivial`, party threshold summation.
- Initiative: full tie-break chain (score desc → dex desc → name asc).
- 404 for unknown routes; 20 concurrent requests all `200`.
