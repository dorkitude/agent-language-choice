Fixed, Chef! 🎲✨

**Root cause:** `src/server.ts` was rounding the dice average with `Math.round`, turning `9.5` into `10` for `1d20-1`.

**Fix:** Changed the average calculation to return the true mean:

```typescript
const average = (min + max) / 2;
```

Verified with a quick local run:

- `GET /health` → `{"ok":true}`
- `1d20-1` → `{"average":9.5}` ✅
- `2d6+3` → `{"average":10}` ✅
- `adjusted-xp` sample → `{"difficulty":"deadly"}` ✅

`./run.sh` is ready and unchanged.
