- `POST /v1/campaigns` → creates a campaign
- `POST /v1/campaigns/:id/characters` → adds a character (404 if campaign missing, 409 if duplicate char ID)
- `POST /v1/campaigns/:id/events` → adds a session log event (404 if campaign missing, 409 if duplicate event ID)
- `GET /v1/campaigns/:id/state` → returns campaign + characters array + `log_count`
