Fixed! 🎯

The bug was in `advanceTurn()`: when the last condition expired, the `conditions` array became empty and PHP serialized it as `[]` instead of `{}`.

**What changed:** `index.php` — after computing combat conditions, if the map is empty, it returns `new stdClass()` so the JSON encoder emits an object.

Verified the failing scenario manually:
```bash
adv5: {"id":"enc-1","round":2,"turn_index":2,"active":{"name":"fighter","score":14},"conditions":{}}
```

`./run.sh` is already in place and unchanged. Server wasn't left running. Ready for evaluation, Chef! 🚀
