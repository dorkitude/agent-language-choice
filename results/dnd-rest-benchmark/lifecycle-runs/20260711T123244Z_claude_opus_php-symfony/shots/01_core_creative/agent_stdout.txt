## What I built 🎲

**`index.php`** — replaced the health-only stub with a full Symfony-components server:
- **Symfony Routing** (`RouteCollection` + `UrlMatcher` + `RequestContext`) dispatches by path *and* method, so wrong methods return a clean 405 and unknown paths return 404.
- **Symfony HttpFoundation** (`Request`/`JsonResponse`) handles I/O.
- Each endpoint is a small pure handler with strict validation: `is_int_like()` rejects bools/strings/non-integral floats, the dice grammar is a single anchored regex, and `normalize_number()` keeps whole numbers as ints (no stray `.0`) while allowing genuine fractions like `9.5`.
- CR keys accept `"1/4"` strings and also coerce numeric `1` → `"1"`.
