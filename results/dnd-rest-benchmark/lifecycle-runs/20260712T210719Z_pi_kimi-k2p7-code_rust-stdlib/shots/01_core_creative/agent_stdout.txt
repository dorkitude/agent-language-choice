Hey Chef! 🎲 The D&D REST engine is ready.

- `src/main.rs` now implements the full stdlib-only server (Rust 1.97.0, no crates).
- `Cargo.toml` is still dependency-free.
- `./run.sh` is preserved: it compiles with `rustc --edition=2024` and runs the server in the foreground.

Endpoints implemented:
- `GET /health` → `{"ok":true}`
- `POST /v1/dice/stats`
- `POST /v1/checks/ability`
- `POST /v1/encounters/adjusted-xp`
- `POST /v1/initiative/order`

I verified the sample payloads (and a few edge cases like `1d20-2`, half-integer multipliers, and tie-breaking) with a temporary `rustc --test` run, then cleaned the tests out to keep the final artifact minimal. I did not start the server. ✨
