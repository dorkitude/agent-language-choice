## Summary

Added three deterministic PHB-rules endpoints to `app.py` (Flask 3.1.3, Python 3.14.6), preserving all prior-stage behavior:

**1. `POST /v1/phb/spell-slots`** — Returns the full-caster slot progression. Supports wizard level 5 → `{"1": 4, "2": 3, "3": 2}`. Rejects unsupported class/level with 400.

**2. `POST /v1/phb/rests/long`** — Applies three long-rest rules:
- HP restored to max
- Hit dice restored by `min(max(level//2, 1), hit_dice_spent)` (half level, min 1, capped by what's spent) → `3 - 2 = 1`
- Exhaustion reduced by 1, floored at 0

**3. `POST /v1/phb/equipment-load`** — Capacity = `strength * 15`; `encumbered` is `weight > capacity` (strictly exceeds, so weight == capacity is *not* encumbered).
