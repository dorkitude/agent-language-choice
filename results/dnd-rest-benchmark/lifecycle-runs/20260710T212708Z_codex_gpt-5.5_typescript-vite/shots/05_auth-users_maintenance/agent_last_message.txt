Implemented Stage 3 auth in `vite.config.ts`.

Added:

- `POST /v1/auth/register`
  - Validates username, password length, and role.
  - Returns `409` for duplicate usernames.
  - Stores only a `crypto.scryptSync` password hash with salt.
- `POST /v1/auth/login`
  - Verifies credentials with timing-safe comparison.
  - Returns deterministic token `session-<username>`.
  - Returns `401` for bad credentials.

`run.sh` is unchanged, executable, and still starts Vite on `127.0.0.1` using `$PORT`.

Verification: `./node_modules/.bin/tsc --noEmit` passed. I did not start the server.
