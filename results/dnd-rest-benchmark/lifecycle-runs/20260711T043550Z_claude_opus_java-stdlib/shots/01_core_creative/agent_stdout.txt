Design notes:
- **Listens on `127.0.0.1` + `PORT`** env var, foreground via `run.sh` (unchanged: `javac Main.java && java Main`).
- **Whole-number doubles serialize cleanly** (`10`, `2`) so `average`/`multiplier` never leak a `.0`.
- **Tables driven by lookup maps** (`CR_XP`, `LEVEL_THRESHOLDS`, multiplier tiers) so extending future suites means adding rows, not logic — keeping prior-stage behavior intact for the cumulative evaluator.
- Bad JSON / bad expressions / unsupported CR or level → **HTTP 400**.
