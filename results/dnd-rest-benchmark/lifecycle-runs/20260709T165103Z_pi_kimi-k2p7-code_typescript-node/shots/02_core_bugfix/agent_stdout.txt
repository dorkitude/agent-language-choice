Fixed! 🎲

The bug was in `src/server.ts` line 57:

```typescript
// before
average: Math.round((dice_count * (1 + sides)) / 2 + modifier),

// after
average: (dice_count * (1 + sides)) / 2 + modifier,
```

For `1d20-1`, the true average is `9.5`, but `Math.round` was snapping it to `10`. The evaluator expects the exact floating-point average.

Verified with `npx tsc --noEmit` — TypeScript check passes. `./run.sh` is unchanged and will compile and start the server in the foreground on `127.0.0.1:$PORT`.
