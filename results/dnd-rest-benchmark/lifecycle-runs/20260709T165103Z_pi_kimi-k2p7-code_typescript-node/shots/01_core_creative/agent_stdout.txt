Created `src/server.ts` implementing the full D&D REST engine using Node 26 built-in `http` and TypeScript 7.0.2:

- `GET /health` → `{"ok": true}`
- `POST /v1/dice/stats` → parses `XdY+Z` expressions, returns count/sides/modifier/min/max/average

- `POST /v1/encounters/adjusted-xp` → CR XP table, count multipliers, summed thresholds, difficulty
- `POST /v1/initiative/order` → score sorting with dex/name tiebreakers
