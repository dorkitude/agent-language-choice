The bug was in `vite.config.ts` where the dice average was being rounded to an integer:

```ts
const average = Math.round((min + max) / 2);
```

For `1d20-1`, min=0, max=19, so the true average is `9.5`, but `Math.round` made it `10`.

Fixed to:

```ts
const average = (min + max) / 2;
```

The existing suite example `2d6+3` still returns `average: 10` exactly, so prior behavior is preserved. TypeScript also checks cleanly.

`./run.sh` is ready and unchanged — it starts Vite on `127.0.0.1:$PORT`.
