## What I added

Three Next.js route handlers under `app/v1/dm/`:

- **`encounter-builder/route.ts`** — looks up each monster's CR from the compendium, reuses the core adjusted-XP math (`CR_XP`, level thresholds, count multiplier) to compute `base_xp`/`adjusted_xp`/`difficulty`, and maps difficulty → a deterministic recommendation. Returns `{base_xp:150, adjusted_xp:300, difficulty:"easy", recommendation:"safe warm-up"}` for the sample. 🎲
- **`loot-parcel/route.ts`** — deterministic tier-keyed loot table (tier 1 → 75gp + 2 healing potions). 💰
- **`session-recap/route.ts`** — pulls the latest campaign event summary and derives open threads from event text (goblin trail → "Resolve goblin trail ambush"). 📜

Plus a small `listEvents` helper in `app/v1/campaigns/store.ts`.
