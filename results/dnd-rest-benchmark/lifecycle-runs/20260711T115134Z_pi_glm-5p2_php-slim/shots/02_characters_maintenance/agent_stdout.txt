**Key implementation notes:**
- `abilityModifier()` uses `(int) floor(($score - 10) / 2)` — not an `(int)` cast (which truncates toward zero), because PHP's int cast on `-0.5` gives `0`, not `-1`. `floor(-0.5)` correctly gives `-1`.
- `proficiencyBonus()` uses `2 + intdiv($level - 1, 4)` — safe since `level >= 1` means no negative-division truncation issues.
- `asInt()` helper accepts JSON ints and whole-number floats but rejects booleans, strings, and fractional floats — keeps validation strict per the "must be an integer" rule.
