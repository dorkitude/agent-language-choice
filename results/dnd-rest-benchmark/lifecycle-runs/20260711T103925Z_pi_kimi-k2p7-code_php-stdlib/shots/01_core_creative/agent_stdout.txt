I created a minimal PHP 8.5.8 stdlib implementation in `index.php` and kept the existing `run.sh`.

What’s implemented:
- `GET /health` → `{"ok": true}`
- `POST /v1/dice/stats` → parses `XdY+/-Z` and returns count, sides, modifier, min, max, average
- `POST /v1/checks/ability` → total, success, margin
- `POST /v1/encounters/adjusted-xp` → base XP, monster count, multiplier, adjusted XP, difficulty, thresholds
- `POST /v1/initiative/order` → sort by score desc, then dex desc, then name asc

I verified the server starts with `run.sh` and all endpoints return the expected sample responses. No Composer packages were added, and the server stays in the foreground using `$PORT` on `127.0.0.1`.
