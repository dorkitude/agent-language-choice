## Summary

I inherited the single-file Rails 8.1.3 API app (`app.rb`, served via `rackup`/`config.ru`) and added the **Monster & Item Compendium** stage while preserving every prior endpoint.

**Changes to `app.rb`:**
- 🗄️ **`Storage` module** — added `MONSTERS` / `ITEMS` in-memory caches, `monsters` + `items` SQLite tables (created in `create_schema`, dropped in `reset`), `save_monster` / `save_item` write-through methods, and cache-loading on boot. Follows the exact durable-storage pattern the existing users/sessions use.
- 🧭 **Routes** — 4 new routes under `/v1/compendium/...`.
- ⚔️ **`CompendiumController`** — create/read for monsters and items.
