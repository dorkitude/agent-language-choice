All set, Chef! 🎲✨

I created `main.go` implementing the full core D&D REST engine with:

- `GET /health` → `{"ok": true}`
- `POST /v1/dice/stats` with regex-parsed expressions
- `POST /v1/checks/ability`
- `POST /v1/encounters/adjusted-xp` with CR/XP tables, multiplier bands, and summed thresholds
- `POST /v1/initiative/order` sorted by score, then dex, then name

I also updated `go.mod` to `1.26.5` and `run.sh` to run the server in the foreground.

I tested it briefly against all the sample requests and the outputs match exactly:

```json
{"ok":true}
{"dice_count":2,"sides":6,"modifier":3,"min":5,"max":15,"average":10}
{"total":14,"success":false,"margin":-1}
{"base_xp":850,"monster_count":3,"multiplier":2,"adjusted_xp":1700,"difficulty":"deadly","thresholds":{"easy":300,"medium":600,"hard":900,"deadly":1600}}
{"order":[{"name":"rogue","score":17},{"name":"ogre","score":15}]}
```

No third-party packages were added — just `net/http`, `encoding/json`, and the standard library.
